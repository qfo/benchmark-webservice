Test Strategy

Select your strategy of testing

Performing the pairwise tests allows you to compare the quality of your predicted orthologs against each selected project independently. Comparisons across the different reference projects are not valid, as the set of proteins in general differs.

Tests on the intersection of several projects allow for a ranking among all the selected projects. Since all the tests are based on the set of common proteins, the power of the test is more and more reduced as the space of possible orthologs gets smaller and smaller the more projects are selected.

Phylogeny-Based Definition Benchmarks


Phylogeny-Based Definition

According to the phylogenetic definition, two homologous genes are orthologs if they diverged through a speciation event. Therefore, the phylogenetic tree of a set of orthologs (a set of genes in which any pair is orthologous) has by definition the same topology as the corresponding species tree.
Altenhoff and Dessimoz, 2009, Phylogenetic and Functional Assessment of Orthologs Inference Projects and Methods PLoS Computational Biology,5(1): e1000262

Function-Based Definition

One of the main application of orthology is the propagation of functional annotation, because orthologs tend to have a similar function.
Altenhoff and Dessimoz, 2009, Phylogenetic and Functional Assessment of Orthologs Inference Projects and Methods PLoS Computational Biology,5(1): e1000262
